# Путешествие по России

# О проекте:
Одностраничный сайт о путешествиях по России.

## Функциональность:

- ссылки на страницы.

## Технологии:

- БЭМ,
- флексбокс-верстка,
- Grid Layout,
- адаптивная верстка (медиазапросы).

## Инструкция по локальному запуску:

- скачать или склонировать репозитрий,
- открыть файл index.html.
